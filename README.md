Abinde is a python module for pygame.

## Advantages

- Easy to use
- Lightweight

## How to use

### Shell

```sh
pip install -U Abinde
```

### Python

```python
import Abinde as ab
game = ab.Game()
player = ab.sprite.Rectangle()
game.mainloop()
```

## Requirements

- __pygame__ for module.
- __python__ for running module.
- __PIL__ for image support
